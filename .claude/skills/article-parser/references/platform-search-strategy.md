# 跨平台搜索策略

## 核心原则

**图片/视频平台链接 → 优先搜索文字版本**

同一作者往往在多平台发布，文字平台更易提取且内容更完整。

## 平台优先级

| 优先级 | 平台类型 | 代表平台 | 提取难度 |
|-------|---------|---------|---------|
| 1 | 文字平台 | 微信公众号、博客、Medium | 简单 |
| 2 | 混合平台 | Twitter/X、知乎 | 中等 |
| 3 | 图片平台 | 小红书 | 困难（需 OCR） |
| 4 | 视频平台 | 抖音、B站 | 困难（需转写） |

## 搜索方法详解

### 1. 搜狗微信搜索（最推荐）

**地址**：https://weixin.sogou.com/

**优点**：
- 覆盖最全的微信公众号索引
- 更新及时
- 支持时间筛选

**搜索技巧**：

| 场景 | 搜索词 | 示例 |
|------|--------|------|
| 知道作者 | `<公众号名> <关键词>` | `数字生命卡兹克 Skills` |
| 知道标题 | 直接搜标题 | `Claude Code 31条秘籍` |
| 技术文章 | `<技术关键词> 教程` | `Agent Memory 实战` |

### 2. Google/Bing 高级搜索

**精确搜索**：
```
"作者名" "标题关键词"
site:mp.weixin.qq.com "关键词"
site:zhihu.com "关键词"
```

**目标平台**：
- `site:mp.weixin.qq.com` - 微信公众号
- `site:zhihu.com` - 知乎
- `site:jianshu.com` - 简书
- `site:medium.com` - Medium
- `site:dev.to` - Dev.to

### 3. 微信 APP 搜一搜

**适用场景**：搜狗未收录的新文章

**步骤**：
1. 打开微信 APP
2. 点击"搜一搜"
3. 输入关键词
4. 选择"文章"分类

### 4. 第三方聚合平台

- **新榜**：https://www.newrank.cn/
- **清博**：https://www.gsdata.cn/

```

A1: 微信公众号/博客/Medium
    → 直接提取（最优情况）

```

## 常见问题

### Q: 什么时候跳过搜索直接提取？

1. 用户明确要求从原平台提取
2. 内容是平台独家（如小红书原创图文）
3. 已知该作者只在该平台发布

### Q: 搜索到多个版本怎么选？

优先级：
1. 微信公众号版本（格式最好）
2. 作者官方博客
3. 内容最完整的版本
4. 发布时间最新的版本

### Q: 搜索效率低怎么办？

- 先提取原链接的作者名和标题
- 用精确匹配搜索（加引号）
- 限制时间范围缩小结果
