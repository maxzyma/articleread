## 背景：什么是 Manus？

Manus 是一个 AI Agent（智能体）开发平台/工具，专注于帮助开发者构建能够自主执行复杂任务的 AI 应用。

---

## 为什么 Claude 适合做 Agent？

基于 Manus 创始人的观点和社区实践，Claude 在 Agent 场景下的优势包括：

### 1. **卓越的编码能力**
- Claude 在代码生成质量方面表现优异
- 在复杂编程任务中稳定可靠
- 相比其他模型至少领先三个版本

### 2. **工具使用能力**
- Agent 需要调用各种工具和 API
- Claude 在工具调用（Tool Use）场景下表现出色
- 能够准确理解和执行多步骤指令

### 3. **开发者生态**
- 主流 AI 编程工具 **Cursor** 大量采用 Claude
- 开发者"用钱投票"，选择 Claude 作为核心模型
- 在实际生产环境中得到验证

### 4. **相对于 GPT 的优势**
- 社区反馈认为 Claude 在过去一年"吊打 GPT"
- 在代码生成、推理能力和指令遵循方面更胜一筹
