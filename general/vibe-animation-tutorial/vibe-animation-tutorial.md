## 概述

Vibe Animation 是一种新兴的动画制作方法，通过 AI 生成代码 + 手势识别实现流畅的矢量动画效果。本教程以"锦鲤跟随手指"为例演示完整流程。

## 技术栈

- **MediaPipe**：手势识别，提取指尖运动轨迹
- **Gemini**（Google AI）：将视觉描述转换为动画代码
- **矢量动画**：非 SVG 格式，基于"鱼骨架运动"（bone-based animation）

## 实现流程

```
1. AI 生成动画描述 → Gemini 生成矢量图形代码
2. 手势录制 → MediaPipe 识别并记录指尖轨迹
3. 代码生成 → 将轨迹数据转换为动画关键帧
4. 渲染输出 → 矢量图形沿轨迹运动
```

## 核心概念

**鱼骨架运动（Bone-based Animation）**：
- 定义矢量图形的骨架结构
- 骨架节点跟随手势轨迹运动
- 图形整体根据骨架形变
